== README

An small Single Page Application coded in Rails + ajax


* Add users

* Create groups 

* Add them manualy

* Or hit the random button to distribute users equitably in all groups

* Sensei role for users in group

* Only one "admin" is possible (he have to delete his account to create a new admin)

NB: i'm not a designer :/  nor a js guy.You may have to reload page(once) to   manualy assign people in group
